# 💬 Real-Time Chat Application

🚀 A full-stack real-time chat application that enables users to connect, add friends, and exchange messages instantly.

🔗 **Live Demo:**
https://realtime-chat-app-phi-blush.vercel.app/

---

## ✨ Features

* 🔐 Secure Authentication (JWT login/register)
* 💬 Real-time messaging using Socket.io
* 👥 Add friends via email/username
* 🟢 Online / Offline user status
* 🔗 Referral / invite code system
* ⚡ Instant message delivery
* 🎨 Multiple UI themes (dark, anime, retro)
* 📱 Responsive UI
* 💾 MongoDB chat storage

---

## 🛠️ Tech Stack

**Frontend:** React.js, Axios
**Backend:** Node.js, Express.js, Socket.io
**Database:** MongoDB
**Auth:** JWT

## ⚙️ Setup

### Clone

git clone https://github.com/ujjawal10024/realtime-chat-app.git

### Backend

cd server
npm install
npm run dev

### Frontend

cd chat-frontend
npm install
npm run dev
